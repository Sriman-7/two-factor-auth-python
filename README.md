# Two-Factor Authentication (2FA) using Python

## Description
This project implements a secure Two-Factor Authentication system using TOTP.

## Features
- Generates secure OTP
- QR code for Google Authenticator
- OTP verification system

## Technologies Used
- Python
- pyotp
- qrcode
- pillow

## How to Run
pip install pyotp qrcode pillow
python two_factor_auth.py

## Output
- QR code generated
- OTP verification successful
